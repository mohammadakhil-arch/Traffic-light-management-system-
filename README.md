# Street Light Monitoring System

An AI-based system designed to monitor street lights, detect faults, and provide real-time insights through a dashboard.

---

 Project Overview

The Street Light Monitoring System is developed to automate the process of detecting faulty street lights using data simulation and machine learning techniques. It helps reduce manual inspection and improves maintenance efficiency.

---

 Features

-  Fault Detection using AI Model
-  Interactive Dashboard for Monitoring
-  Data Simulation for Testing
-  Alerts for Faulty Street Lights
-  Modular Code Structure

---

Technologies Used

- Python
- Machine Learning
- Flask (Dashboard)
- Data Simulation
- Git & GitHub

---

 Project Structure

street-light-monitoring/
│── ai_model/        # Machine learning model
│── dashboard/       # Web dashboard (Flask)
│── simulator/       # Data simulation
│── utils/           # Utility functions
│── main.py          # Main execution file
│── requirements.txt # Dependencies

---

Installation & Setup

1. Clone the repository:

git clone https://github.com/mohammadakhil-arch/Traffic-light-management-system-

2. Navigate to the project folder:

cd street-light-monitoring

3. Install dependencies:

pip install -r requirements.txt

---

How to Run

1. Run the simulator:

python simulator/simulator.py

2. Run the dashboard:

python dashboard/app.py

3. Open browser and go to:

http://localhost:5000
